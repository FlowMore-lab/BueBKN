# 🛡 BueRKN - Anti-DPI Tool

**Обход блокировок DPI (Deep Packet Inspection) для доступа к заблокированным сервисам**

## 📌 Возможности

- Обход DPI через фрагментацию и маскировку трафика
- Поддержка всех популярных сервисов:
  - Telegram
  - WhatsApp
  - Instagram
  - Twitter / X
  - YouTube
  - Discord
- Автоматическая смена DNS
- Режим `chaos` для максимальной обфускации

---

## 🚀 Быстрый старт

### 1. Cкачай этот репозиторий и распакуй на рабочий стол (без кирилицы в пути).

### 2. Запусти Utils/dns_fix.bat, для того чтоб резволить все зарубежные домены и ip.

### 3. Запусти Utils/get_all_ips.bat, для того чтоб резволить все зарубежные домены и ip (ЛОКАЛЬНО).

### 4. Теперь все готово к запуску, запускай BueRKN.bat и смейся над Роскомнадзором😂!
